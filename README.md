# LP Mundo Animal · Clínica Veterinária 24h

Landing page de conversão da **Mundo Animal — Clínica Veterinária 24h** (Goiânia/GO).
Página única em HTML + CSS + JS (sem frameworks), responsiva e otimizada para desktop e mobile.

## Estrutura

```
.
├── index.html              # Página completa (HTML + CSS + JS inline)
└── assets/
    ├── hero.png            # Imagem de fundo do hero (desktop)
    ├── hero-mobile.png     # Imagem de fundo do hero (mobile)
    └── carrossel/          # Fotos da estrutura (01.jpg … 10.jpg)
```

## Destaques

- Hero com bifurcação de públicos: **emergência 24h** (ligação direta) e **agendamento de rotina**.
- CTAs com intenção variada por seção.
- Cards de serviço, pilares de diferenciais e carrossel de estrutura.
- Micro-animações ao rolar, contadores animados e selo "atendimento ativo agora".
- Tipografia Figtree; identidade em verde/teal com alerta em coral para emergência.

## Publicação

Servida via **GitHub Pages** a partir do `index.html` na raiz.
Para rodar localmente, basta abrir o `index.html` no navegador (ou servir a pasta com qualquer servidor estático).

> Observação: a seção de depoimentos usa textos placeholder até a inserção das avaliações reais.
